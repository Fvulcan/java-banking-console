## Java Banking Console

This is my first Java project. (First Version)

Java feels a bit strict sometimes and honestly a little tricky at the beginning, but I still like working with it. This project was a good way for me to get more comfortable with the basics.

What this project does

This is a simple banking app that runs in the console.

The user can:

* check the current balance
* deposit money
* withdraw money
* view the transaction history
* exit the program

The program also checks invalid amounts and prevents withdrawals if the balance is not high enough.


## Current Version

I extended the project with SQLite.

The data is now stored in a local database instead of only being kept in memory.
